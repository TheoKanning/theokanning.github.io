## How does it work?

When you take a picture, it's sent to theÂ [Google Cloud Vision Api](https://cloud.google.com/vision), which returns a list of potential labels. Once the app has a list of labels, it compares them to a local emoji map, or uses [emoji-java](https://github.com/vdurmont/emoji-java)Â if the labels aren't in the local map.

The main difficulty is that Google's labels rarely correspond to emojis. To fix this, I send a firebase analytics event each time a label isn't recognized. When I check all of the events later, I just add it to the map. It's a pretty manual process, but it didn't take long to add a good base list of emojis.
